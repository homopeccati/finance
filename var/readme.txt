Калькулятор VaR рассчитывает показатель 10 дневного Value-at-Risk по портфелю опционов с использованием симуляций Монте-Карло при допущении, что цены на акции следуют Геометрическому Броуновскому Движению (GBM).

Архив содержит три файла .py и файл .csv, содержащий данные по опционам (в частности тикеры на срочной секции Московской Биржи и количество опционов в портфеле, где число меньше нуля обозначает короткую позицию по инструменту). 

Описание .py файлов:
- var.py - пользователь предоставляет на вход дату, для которой рассчитывается VaR.
- data_parser.py - вспомогательный файл, используемый основным скриптом. Содержит функции для выгрузки данных с МосБиржи.
- config.py - файл, содержащий параметры конфигурации симуляции Монте-Карло. Может быть изменен в части количества дней для симуляции цен активов или в части количества симуляций.
